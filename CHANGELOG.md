# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-13

### Added
- Initial release of thread-blocker
- `blockMainThread()` - Async function to block the main thread
- `blockMainThreadSync()` - Synchronous blocking function
- `block()` - Advanced blocking with options
- `blockRepeatedly()` - Repeated blocking patterns
- `getBlockingInfo()` - Function to get blocking capabilities
- Full TypeScript support with type definitions
- Comprehensive README with examples
- Example usage file

### Features
- CPU-intensive and low-intensity blocking modes
- Optional callbacks after blocking
- Configurable intervals between repeated blocks
- Promise-based API for async operations
